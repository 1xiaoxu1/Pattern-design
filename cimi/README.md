# Pattern-ui

## Getting Started

- 安装

```bash
$ npm i Pattern-ui -D
```

- 使用

```js
import { Button } from 'Pattern-ui'
import 'Pattern-ui/dist/index.css'
function App(){
    return (
        <Button>测试</Button>
    )
}

```
```bash
$ npm start
```

Build documentation,

```bash
$ npm run docs:build
```

Run test,

```bash
$ npm test
```

Build library via `father`,

```bash
$ npm run build
```
